# React Documentation PDF

A downloadable PDF version of the official React.js documentation for offline reading and reference.

## Download

Get the latest PDF from the [Releases](https://github.com/TopMyster/React-Documentation-PDF/releases/tag/pdf) page:

- [**reactjsdocs.pdf**](https://github.com/TopMyster/React-Documentation-PDF/releases/download/pdf/reactjsdocs.pdf) (67.8 MB)

## Why?

- Read React docs on a plane, commute, or anywhere without internet
- Search and annotate with your favorite PDF reader
- Keep a local copy for quick reference

## Contents

The PDF covers the full React documentation, including:

- Getting Started
- Core Concepts (Components, JSX, Props, State)
- Hooks
- Advanced Guides
- API Reference
